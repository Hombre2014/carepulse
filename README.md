# CarePulse

> A healthcare app for managing patients' appointments.

## Built With

- Major languages: TypeScript
- Backend: Appwrite.io
- Framework: Next.js
- Tools used: Twilio, Sentry
- Styling: TailwindCSS, shadcn/ui
